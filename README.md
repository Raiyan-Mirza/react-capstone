# 🎮 Gamified Task Manager

A modern **React-based productivity app** that turns your daily tasks into a game.  
Complete tasks, earn XP, level up, and stay motivated while managing your goals.


## 📌 Features

- ✅ Add and manage daily tasks
- 🎯 Gain XP for completing tasks
- 📈 Progress bar system
- 🏆 Level-up motivation mechanics
- 🗑️ Delete completed/unwanted tasks
- 🌙 Clean dark UI
- ⚡ Fast and responsive design

## 🛠️ Built With

- React.js
- JavaScript
- CSS 
- Vercel (Deployment)
